# MCHOSE Ace 68 - SignalRGB Plugin ⌨️✨

A built-from-scratch, fully reverse-engineered SignalRGB plugin for the [**MCHOSE Ace 68**](https://www.mchose.store/products/mchose-ace-68-hall-effect-magnetic-switch-gaming-keyboard) keyboard. 

This plugin seamlessly integrates the keyboard into [SignalRGB](https://signalrgb.com/), allowing full synchronization of your RGB lighting effects with the rest of your setup.

## 🚀 Features
* **Perfect 65% Layout:** All 68 keys are exactly mapped and assigned to the lighting matrix.
* **Plug & Play:** Immediately recognized by SignalRGB as a keyboard matrix.

## 📦 Installation

Installation is super easy and can be done directly via the Add-on Manager in SignalRGB:

1. Open **SignalRGB**.
2. Click the **Gear icon (Settings)** in the bottom left corner.
3. Select **Add-ons** from the left menu.
4. Click the **+ Add-on** button.
5. Paste the following GitHub link into the field:
   [https://github.com/Luk-Krn/mchose_ace68.git](https://github.com/Luk-Krn/mchose_ace68.git)
